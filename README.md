# binconv
Handy tool to convert between binary and text

## Building

```sh
mvn clean package
```

## Usage

```sh
java -jar binconv.jar -<MODE> [SRC] -i <SRC_FILE_PATH> -o <DEST_FILE_PATH> [-newline <NEWLINE_POS>]
```
MODE: frombase64|tobase64|fromhex|tohex|frombin|tobin

## Example
From Base64 text file to binary file

```sh
java -jar binconv.jar -frombase64 -i base64.txt -o image.jpg
```

From binary file to Base64 text file

```sh
java -jar binconv.jar -tobase64 -i image.jpg -o base64.txt
```
